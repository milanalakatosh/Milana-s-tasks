Собираем проект:

$ bem make

Первая строка с bem tools займет какое-то время, так как именно в это время в директорию устанавливаются дополнительные npn-пакеты.

Для запуска сервера выполняем:

bem server

В результате вы увидете следующее сообщение:

Info: server is listening on port 8080. Point your browser to/ htpp://local host:8080....

Проблема?

Если порт уже используется другой программой, его можно переназначить с помощью аргумента -p.
